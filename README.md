# DEV-OPS-LR5 (Ansible — Работа с переменными и запуск Nmap)
Этот репозиторий содержит реализацию лабораторной работы по использованию переменных в Ansible для установки и запуска сканера Nmap на целевых хостах, сбора и вывода результатов сканирования.

## Описание задания
Плейбук должен на целевых хостах:
 - установить Nmap;
 - скопировать файл со списком целей для сканирования (`targets.txt`);
 - запустить Nmap для этих целей;
 - собрать результаты сканирования (stdout);
 - вывести результаты сканирования.

Подготовка окружения аналогична предыдущей лабораторной (настройка виртуальных машин, SSH-доступ, inventory и т.п.).

## Содержимое репозитория
 - `NMap.yml` — один из вариантов Ansible-плейбука для установки Nmap и запуска сканирования.  
 - `loopNMap.yml` — альтернативный вариант плейбука, использующий цикл (loop) для последовательного сканирования целей.  
 - `targets.txt` — файл со списком целей/хостов для сканирования (одна цель в строке).  

## Используемые технологии
 - Ansible — управление конфигурацией и выполнение задач на удалённых хостах.  
 - Nmap — инструмент для сетевого сканирования.  
 - SSH — для удалённого подключения к целевым машинам.
